JavaScript Lovers

We at AppsAttic not only know JavaScript but we like it too! Client side JS such as jQuery (our library of choice) we know inside out and jQueryUI is a great complement. Newer tools such as underscore.js, socket.io and backbone.js are also firm favourites here.

NodeJS Winners

When we see a disruptive technology like NodeJS we pounce and build great stuff like CSSMinifier. Using the powerhouses of Node, we plug in to the awesome sauce that is connect, Express, everyauth, , and our very own node-awssum. Really, you should check it out!

Cloud Web API Masters

Having written API Libraries to lots (and lots, and lots) of Web APIs, we know a thing or two about talking to your service, or the services you depend on. With almost 100 GitHub watchers, node-awssum is a great library to make your app talk to other sites. Talk to us to find out more!

Open Source People

Open source is a part of our pysche (npm, AppsAttic on GitHub, Chilts on GitHub, cpan). We've used it for years, we've contributed code for years and we've enjoyed it all the way. Giving back to the community just makes us feel better ... there, we've said it ... now GET OUTTA HERE and go look!
